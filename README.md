
# HR Analytics Excel Dashboard

📌 Project Overview
This project presents an interactive HR Analytics Dashboard built using Microsoft Excel. The dashboard provides insights into employee attrition, workforce distribution, salary analysis, job satisfaction, overtime trends, and work-life balance to support data-driven HR decision-making.

🎯 Project Objectives
- Analyze employee attrition trends.
- Monitor workforce distribution across departments and job roles.
- Track key HR metrics using KPI cards.
- Create an interactive dashboard using PivotTables, PivotCharts, and Slicers.
- Generate business insights and recommendations.

🛠️ Tools Used
- Microsoft Excel
- PivotTables
- PivotCharts
- Slicers
- Conditional Formatting
- Excel Formulas

 📊 Key Performance Indicators (KPIs)
- Total Employees: **1470**
- Attrition Count: **237**
- Attrition Rate: **16.12%**
- Average Monthly Income: **6502.93**
- Highest Monthly Income: **19999**
- Lowest Monthly Income: **1009**
- Average Age: **36.9 Years**
- Average Years at Company: **7.0 Years**

 📈 Dashboard Features
- Employee Distribution by Department
- Employee Distribution by Job Role
- Attrition by Years at Company
- Overtime vs Attrition
- Job Satisfaction vs Attrition
- Work-Life Balance vs Attrition
- Gender Distribution
- Average Monthly Income by Job Role
- Interactive Slicers for filtering dashboard data

 💡 Key Insights
- The organization has an attrition rate of **16.12%**.
- A total of **237 employees** have left the company.
- Overtime and employee satisfaction show a relationship with attrition.
- Workforce distribution and salary analysis help identify HR trends.
- Interactive filters allow dynamic analysis across departments, gender, job role, and overtime.

💼 Business Recommendations
- Improve employee satisfaction initiatives.
- Monitor overtime to reduce employee burnout.
- Focus retention strategies on high-attrition groups.
- Review compensation for key job roles.
- Enhance employee engagement and career development programs.
